# Syntax and Variables

## Syntax

     ┌───────────── min (0 - 59)
     │ ┌────────────── hour (0 - 23)
     │ │ ┌─────────────── day of month (1 - 31)
     │ │ │ ┌──────────────── month (1 - 12)
     │ │ │ │ ┌───────────────── day of week (1 - 7, or mon - sun)
     │ │ │ │ │
     │ │ │ │ │
     * * * * *  command-to-execute --arguments ...  # comment documentation

Note that times are 0-based, but dates are 1-based, just like the real world.
